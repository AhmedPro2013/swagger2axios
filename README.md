# swagger2axios

install dependencies
```
npm install
```

configuration
```yaml
edit bootstrap.yml file

put your swagger api address in 'url' field
'name' field is the name of the folder where your code will be generated
```

run
```
node indexold.js  
```

done
```
find your file in target folder
```

have fun!
